# Library Management System

## Author
Abdelrahman Ahmed Elkhmisi

## Project Description

Library Management System is a Python application connected to an SQL database.
The system allows users to manage books, members, and borrowing records.

The project supports both:
- Graphical User Interface (GUI)
- Command Line Interface (CLI)

## Technologies Used

- Python
- SQLite Database
- Tkinter GUI
- GitHub

## Features

### Book Management
- Add new books
- View all books
- Delete books

### Database Management
The application stores data using an SQLite database.

Database tables:

- Books
- Members
- Borrowing Records
